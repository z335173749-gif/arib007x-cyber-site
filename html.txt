<h2>📘 HTML - Structure of Websites</h2>

<p>
HTML (HyperText Markup Language) is used to create the structure of a website.
It tells the browser what to display.
</p>

<h3>🏷️ Important Tags</h3>
<ul>
  <li>&lt;h1&gt; to &lt;h6&gt; → Headings</li>
  <li>&lt;p&gt; → Paragraph</li>
  <li>&lt;a&gt; → Link</li>
  <li>&lt;img&gt; → Image</li>
  <li>&lt;div&gt; → Container</li>
  <li>&lt;button&gt; → Button</li>
</ul>

<h3>💻 Example (Real Structure)</h3>

<pre>
&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
  &lt;title&gt;My Page&lt;/title&gt;
&lt;/head&gt;

&lt;body&gt;

  &lt;h1&gt;Welcome to My Website&lt;/h1&gt;

  &lt;p&gt;This is a paragraph of text.&lt;/p&gt;

  &lt;a href="https://google.com"&gt;Go to Google&lt;/a&gt;

&lt;/body&gt;
&lt;/html&gt;
</pre>

<h3>⚡ Tip</h3>
<p>HTML is NOT programming — it is markup used to build structure.</p>
